## Summary

NSE will conduct a mock trading session in the Securities Lending and Borrowing Market (SLBM) on Saturday, August 30, 2025. The session will run from 09:15 to 10:10 for trading, followed by re-login testing from 13:30 to 14:00. No new software version will be released for this session.

## Key Points

- Mock trading date: Saturday, August 30, 2025
- Trading session: 09:15 to 10:10
- Re-login testing: 13:30 to 14:00
- Software version: NEAT 1.3.3 (no new release)
- All outstanding orders will be cleared before each session
- No financial obligations - trades will not result in fund pay-in or pay-out
- NOTIS application will not be accessible during mock session
- Connect2NSE and Extranet facility will remain available

## Regulatory Changes

No regulatory changes announced. This is a routine mock trading session using existing NEAT 1.3.3 software version.

## Compliance Requirements

- Members must use valid and compliant UCC/PAN uploaded and approved before cutoff
- NNF software users must manually clear orders in their systems
- Members using Non-NEAT Frontend must ensure User ID is dealer type, converted for NNF, and mapped with IP
- Set appropriate branch and user limits from Corporate Manager Terminal before placing orders
- Members should participate using all trading software and re-login into live environment

## Important Dates

- **August 30, 2025**: Mock trading session
  - 09:15-10:10: Trading session from DR Site
  - 13:30-14:00: Live re-login testing
- **September 1, 2025**: Normal trading resumes on Monday

## Impact Assessment

Medium operational impact as this is a scheduled mock trading session designed to ensure system readiness and connectivity testing for SLBM participants. Members are encouraged to participate to ensure smooth operations for the following Monday. The session provides opportunity to test connectivity, order management systems, and re-login procedures without financial obligations.

## Summary

NSE will conduct a mock trading session in the Capital Market Segment on Saturday, August 23, 2025. The session includes various trading windows from 10:45 to 14:30 hours, with no new software version release. All capital market members are automatically eligible to participate using existing credentials.

## Key Points

- Mock trading date: August 23, 2025
- Trading sessions run from 10:45 to 14:30 hours
- No new version release - current NEAT+ 7.8.3 to be used
- All outstanding orders will be cleared before each session
- No financial obligations - trades will not result in fund pay-in or pay-out
- NOTIS application will not be accessible during mock session
- Connect2NSE and Extranet facility will remain available

## Regulatory Changes

No regulatory changes announced. This is a routine mock trading session for system testing and member preparation.

## Compliance Requirements

- Members must ensure migration to NEAT+ version 7.8.3 before September 6, 2025 discontinuation of version 7.8.2
- Only valid and compliant UCC/PAN uploaded and approved before cutoff will be allowed
- Members using NNF software must manually clear orders in their systems
- Members should participate using all trading software and re-login into live environment

## Important Dates

- **August 23, 2025**: Mock trading session
- **August 25, 2025**: Return to normal live trading (Monday)
- **September 6, 2025**: Discontinuation of NEAT+ version 7.8.2

## Impact Assessment

Medium operational impact as all capital market members need to participate in mock trading to ensure system readiness. Session designed to test connectivity and software functionality before normal trading resumes on Monday. No market disruption expected as this is a scheduled weekend activity.

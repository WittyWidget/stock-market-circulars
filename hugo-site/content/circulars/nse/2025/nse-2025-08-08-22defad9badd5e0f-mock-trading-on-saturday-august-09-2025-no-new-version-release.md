## Summary

NSE will conduct mock trading sessions in the Currency Derivatives segment on Saturday, August 09, 2025. The session includes trading from both Primary Site (11:00-15:00) and DR Site (15:30-16:15), followed by live re-login (18:00-18:30). No new software version will be released.

## Key Points

- Mock trading scheduled for Saturday, August 09, 2025
- Two trading sessions: Primary Site (11:00-15:00) and DR Site (15:30-16:15)
- Live re-login session from 18:00-18:30
- Software version remains NEAT 3.5.3
- All outstanding orders will be cleared before each session
- No financial obligations for trades executed during mock sessions

## Trading Schedule

- **Primary Site Trading**: 11:00 - 15:00
- **DR Site Trading**: 15:30 - 16:15
- **Trade Modification End**: 16:25
- **Live Re-login**: 18:00 - 18:30

## Technical Requirements

- Only valid and compliant UCC/PAN will be allowed
- Members must use existing User IDs, IPs, and Box ID mappings
- NNF software users must manually clear orders
- NOTIS application will not be accessible during mock session
- Connect2NSE and Extranet limited availability (12:00 PM - 11:00 PM)

## Important Notes

- All CD segment members automatically eligible for mock trading
- UCC validation skipped during contingency time
- Members should participate using all trading software
- Live updates available at www.nseindia.com
- Support available at Toll-Free: 1800 266 0050 (Option 1)

## Impact Assessment

Medium impact event focused on system testing and member preparation. Essential for ensuring smooth trading operations on Monday, August 11, 2025. Members are encouraged to participate to verify connectivity and system readiness.

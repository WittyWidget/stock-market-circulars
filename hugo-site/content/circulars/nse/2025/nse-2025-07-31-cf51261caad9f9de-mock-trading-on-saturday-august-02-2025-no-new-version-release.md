## Summary

NSE will conduct a mock trading session for the Commodity Derivatives segment on Saturday, August 02, 2025. The session will test new pre-trade risk controls for pre-emptive order cancellation on account of NNF ID-Algo ID validations, which will go live from August 04, 2025.

## Key Points

- Mock trading scheduled for Saturday, August 02, 2025
- No new software version release - continues with NEAT 1.5.1
- Two trading sessions: Primary site (11:00-14:00) and BCP site (14:45-15:30)
- Testing of new pre-trade risk controls for NNF ID-Algo ID validations
- All outstanding orders will be cleared before each session
- No financial obligations from mock trades

## Regulatory Changes

Introduction of pre-trade risk controls with pre-emptive order cancellation on account of NNF ID-Algo ID validations, effective from August 04, 2025 in live trading.

## Compliance Requirements

- Members must participate using all trading software
- Members must re-login into live environment after mock session
- Only valid and compliant UCC/PAN will be allowed (validation skipped during contingency time)
- Members using NNF software must manually clear orders in their systems
- Compliance with SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234 for software testing

## Important Dates

- **August 02, 2025**: Mock trading session
  - Session 1 (Primary): 11:00-14:00
  - Session 2 (BCP): 14:45-15:30
  - Live Re-login: 17:30-18:00
- **August 04, 2025**: New pre-trade risk controls go live

## Impact Assessment

Medium impact routine system testing session. Members need to ensure participation to test connectivity and new risk control features before they become effective in live trading. The new pre-trade risk controls will affect order processing for algorithmic trading participants.

## Summary

NSE will conduct a mock trading session in the Currency Derivatives Segment on Saturday, October 11, 2025. This session is related to the new encryption mechanism of Interactive Messages and Immediate Order acknowledgement. No new version will be released during this session. Members are required to participate using all trading software and re-login into the live environment.

## Key Points

- Mock trading session scheduled for October 11, 2025 in Currency Derivatives Segment
- Primary Site Trading: 11:00 to 13:30 hrs
- DR Site Trading Session-2: 14:00 to 14:30 hrs (Trade Modification end: 14:40 hrs)
- Live Re-login: 16:00 to 16:30 hrs
- NEAT Version 3.5.3 will be discontinued on November 01, 2025 (mock)
- NEAT Version 3.5.4 is recommended and available
- New NEAT Adapter Version 1.0.22 (Windows and Linux) uses port 10879
- Old NEAT Adapter Version 1.0.20 uses port 10877
- All outstanding orders will be cleared before each session
- No financial obligations will arise from mock trades
- NOTIS application will not be accessible during mock session
- Connect2NSE and Extranet facility will be available

## Regulatory Changes

No regulatory changes introduced. This circular relates to operational testing of the new encryption mechanism announced in earlier circular NSE/CD/70422 dated September 25, 2025.

## Compliance Requirements

- Members must participate in the mock trading session
- Members must use either NEAT Version 3.5.3 or 3.5.4 (3.5.4 recommended)
- Members should ensure appropriate NEAT Adapter version and port configuration:
  - Old Adapter (1.0.20): Use port 10877
  - New Adapter (1.0.22): Use port 10879
- Members using NNF software must manually clear orders in their systems
- Only valid and compliant UCC/PAN uploaded and approved before cutoff will be allowed
- Members should re-login into live environment after mock session to ensure smooth connectivity for Monday, October 13, 2025
- UCC Validation will be skipped during contingency time for order entry

## Important Dates

- **October 10, 2025**: Circular issue date
- **October 11, 2025**: Mock trading session date
  - 11:00-13:30 hrs: Primary Site Trading
  - 14:00-14:30 hrs: DR Site Trading Session-2
  - 14:40 hrs: Trade Modification end time
  - 16:00-16:30 hrs: Live Re-login window
- **November 01, 2025**: NEAT Version 3.5.3 discontinuation date (mock)

## Impact Assessment

**Operational Impact**: Medium - Members must participate in the mock session to test system compatibility with new encryption mechanism and ensure readiness for live trading on October 13, 2025.

**Technical Impact**: Members need to verify their NEAT version compatibility and NEAT Adapter configuration. Those still using Version 3.5.3 should plan migration to 3.5.4 before November 01, 2025.

**Financial Impact**: None - No financial obligations arise from mock trading activities.

**Member Action Required**: Active participation in mock trading session, verification of software versions, testing of connectivity through both Primary and DR sites, and successful re-login to live environment post-session.

## Summary

NSE will conduct a mock trading session for the Commodity Derivatives segment on Saturday, August 23, 2025. The session will run from 11:00 to 12:00 hours with no new software version release. This is a routine system testing exercise to ensure smooth operations for the following trading day.

## Key Points

- Mock trading date: Saturday, August 23, 2025
- Trading session timing: 11:00 to 12:00 hours
- Segment: Commodity Derivatives only
- No new software version release
- Position limit/collateral setup cutoff: 12:10
- Live re-login window: 14:00 to 14:30
- Software version: NEAT 1.5.1

## Technical Requirements

- All CD segment members automatically eligible
- Use existing User IDs, IPs, and Box ID mappings
- Outstanding orders will be cleared before session
- NNF software users must manually clear orders
- NOTIS application will not be accessible
- Connect2NSE and Extranet remain available

## Compliance Requirements

- Only valid and compliant UCC/PAN will be allowed
- Members must set appropriate branch and user limits
- Participation recommended using all trading software
- Members should re-login to live environment post-session

## Important Dates

- **August 23, 2025**: Mock trading session (11:00-12:00)
- **August 25, 2025**: Regular trading resumes on Monday

## Impact Assessment

Routine operational exercise with no financial obligations. Members should participate to ensure connectivity and system readiness for Monday trading. No market impact expected as this is a testing environment only.

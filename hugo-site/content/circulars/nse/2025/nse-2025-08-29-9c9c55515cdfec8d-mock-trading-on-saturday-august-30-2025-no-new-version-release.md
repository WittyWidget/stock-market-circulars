## Summary

NSE will conduct a mock trading session in the Commodity Derivatives Segment on Saturday, August 30, 2025. This is a routine testing session with no new version release, designed to ensure system connectivity and functionality before the next trading day.

## Key Points

- Mock trading session scheduled for Saturday, August 30, 2025
- Trading session: 09:00 to 10:10 hours
- Market close time: 10:20 hours
- Position limit/collateral setup cutoff: 10:20 hours
- Live re-login window: 13:30 to 14:00 hours
- No new software version release
- Using NEAT 1.5.1 software version
- All outstanding orders will be cleared before session

## Regulatory Changes

No regulatory changes announced. This is a routine mock trading session.

## Compliance Requirements

- Members must participate using all trading software
- Must re-login into live environment to ensure smooth connectivity for Monday, September 1, 2025
- Only valid and compliant UCC/PAN will be allowed
- NNF software users must manually clear orders in their systems
- Members using NNF must ensure User ID is dealer type, converted for NNF, and mapped with IP
- Set appropriate branch and user limits before placing orders

## Important Dates

- **August 30, 2025**: Mock trading session
- **September 1, 2025**: Next regular trading day
- Trading hours: 09:00 - 10:10
- Re-login window: 13:30 - 14:00

## Impact Assessment

Minimal market impact as this is a testing session with no financial obligations. No fund pay-in or pay-out will result from trades during the mock session. Members should use this opportunity to test connectivity and systems before regular trading resumes on Monday.

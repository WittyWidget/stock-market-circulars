## Summary

NSE will conduct a mock trading session in the Commodity Derivatives segment on Saturday, August 09, 2025. The session includes two trading periods - one from the primary site and another from the disaster recovery site, with no new software version release.

## Key Points

- Mock trading date: August 09, 2025
- Two trading sessions: Primary site (11:00-15:00) and DR site (15:30-16:15)
- Software version: NEAT 1.5.1 (no new release)
- All outstanding orders cleared before each session
- Live re-login window: 18:00-18:30
- No financial obligations from mock trades

## Trading Schedule

- **Session 1 (Primary Site)**: 11:00 - 15:00
- **Session 2 (DR Site)**: 15:30 - 16:15
- **Position Limit Setup**: 16:25
- **Trade Modification End**: 16:25
- **Live Re-login**: 18:00 - 18:30

## System Requirements

- Only valid and compliant UCC/PAN allowed
- Members must use existing User IDs, IPs, and Box ID mappings
- NNF users require dealer-type User IDs
- NOTIS application unavailable during mock session
- Extranet/Connect2NSE limited availability (12:00 PM - 11:00 PM)

## Important Guidelines

- All CD segment members automatically eligible
- Members should test all trading software
- Mandatory re-login to live environment post-session
- UCC validation skipped during contingency periods
- Support available via toll-free: 1800 266 0050 (Option 1)

## Impact Assessment

Routine system testing with minimal market disruption. Members required to participate to ensure smooth operations for Monday, August 11, 2025 trading.

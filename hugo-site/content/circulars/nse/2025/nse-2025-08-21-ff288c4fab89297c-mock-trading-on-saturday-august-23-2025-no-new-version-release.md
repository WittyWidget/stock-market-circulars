## Summary

NSE will conduct a mock trading session in the Currency Derivatives Segment on Saturday, August 23, 2025. This is a routine testing session with no new software version release. All members enabled for Currency Derivatives trading are automatically eligible to participate.

## Key Points

- Mock trading scheduled for Saturday, August 23, 2025
- Trading session: 11:00 to 12:00 hrs
- Trade modification end time: 12:10 hrs
- Live re-login window: 14:00 to 14:30 hrs
- No new software version release
- Current software version: NEAT 3.5.3
- No financial obligations from mock trades
- NOTIS application will not be accessible during session

## Regulatory Changes

No regulatory changes announced in this circular.

## Compliance Requirements

- Members must use valid and compliant UCC/PAN uploaded before cutoff
- NNF users must manually clear outstanding orders in their systems
- Members should set appropriate branch and user limits before placing orders
- All outstanding orders will be cleared before each session
- UCC validation will be skipped during contingency time

## Important Dates

- **August 23, 2025**: Mock trading session date
- **11:00-12:00 hrs**: Trading session window
- **12:10 hrs**: Trade modification end time
- **14:00-14:30 hrs**: Live re-login window
- **August 25, 2025**: Next regular trading day (Monday)

## Impact Assessment

This is a routine mock trading session with minimal market impact. The session serves as a system test to ensure smooth connectivity for regular trading. Members are encouraged to participate to verify their trading systems and connectivity. No financial settlements will occur from mock trades.

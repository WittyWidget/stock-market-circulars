## Summary

NSE will conduct a mock trading session in the Securities Lending and Borrowing Market (SLBM) segment on Saturday, October 11, 2025. The session will include trading from both Primary and DR (Disaster Recovery) sites, followed by live re-login testing. No new software version will be released. This is a routine testing exercise with no financial obligations arising from trades executed during the session.

## Key Points

- Mock trading scheduled for October 11, 2025 in SLBM segment
- Trading Session-1 from Primary Site: 11:15 to 13:30
- Trading Session-2 from DR Site: 14:15 to 14:30
- Live Re-login window: 16:00 to 16:30
- Software version: NEAT 1.3.3 (no new version release)
- All outstanding orders will be cleared before each session
- No financial obligations from mock trades
- All members enabled for SLBM trading in live market are automatically eligible
- Connect2NSE and Extranet will be unavailable from 09:00 pm to 12:00 am

## Regulatory Changes

No regulatory changes introduced. This circular relates to routine mock trading procedures.

## Compliance Requirements

- **UCC/PAN Validity**: Only valid and compliant UCC/PAN uploaded and approved before cutoff will be allowed for order entry (as per circulars NSE/ISC/51754 and NSE/ISC/52722)
- **Software Testing**: Members using NNF software must manually clear orders in their systems
- **Branch/User Limits**: Set appropriate limits from Corporate Manager Terminal before placing orders
- **NNF Application Users**: User ID must be dealer type, converted for NNF, and mapped with IP
- **Box ID Setup**: Ensure Box ID is created on IP with appropriate messages for SLBM segment
- **Telnet Connectivity**: Verify successful telnet connection from intended IP address
- **Participation Recommended**: Members should participate using all trading software and re-login to ensure smooth connectivity for Monday, October 13, 2025

## Important Dates

- **October 10, 2025**: Circular issue date
- **October 11, 2025**: Mock trading session date
  - 11:15-13:30: Trading Session-1 (Primary Site)
  - 14:15-14:30: Trading Session-2 (DR Site)
  - 16:00-16:30: Live Re-login window
- **October 13, 2025**: Next trading day (Monday) - members should ensure connectivity readiness

## Impact Assessment

**Market Impact**: None. This is a mock trading session with no financial obligations.

**Operational Impact**: Low. Routine testing session to verify system readiness and connectivity. Members are encouraged to participate to ensure smooth operations on the following trading day. The session tests both primary and disaster recovery site functionality.

**Member Action Required**: Optional but recommended participation to test connectivity and systems. Members should verify all prerequisites including UCC/PAN validity, telnet connectivity, Box ID setup, and branch/user limits before the session.

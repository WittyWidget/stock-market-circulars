## Summary

NSE will conduct a mock trading session in the Currency Derivatives Segment on Saturday, September 06, 2025. Members must use NEAT Adapter version 1.0.20 and participate in both primary site and disaster recovery site sessions to ensure smooth connectivity for Monday trading.

## Key Points

- Mock trading scheduled for September 06, 2025 in Currency Derivatives segment
- Two trading sessions: Primary Site (11:00-14:15) and DR Site (15:00-15:45)
- Members must revert to NEAT Adapter version 1.0.20 from version 1.0.21
- Live re-login session from 18:00-18:30
- No financial obligations from mock trades
- All outstanding orders cleared before each session

## Trading Schedule

**Primary Site Session:**
- Trading Session-1: 11:00-14:15
- Normal Market: 11:00-12:00
- Contingency: 12:00-13:30
- Market Close: 13:30-14:15

**DR Site Session:**
- Trading Session-2: 15:00-15:45
- Trade Modification: Until 15:55
- Live Re-login: 18:00-18:30

## Technical Requirements

- Use NEAT Adapter version 1.0.20 (Windows/Linux)
- Login through GR PORT 10877
- NEAT 3.5.3 software version
- Valid UCC/PAN compliance required
- Manual order clearing for NNF software users

## System Availability

- Connect2NSE/Extranet unavailable 04:00-17:00
- Extranet API unavailable 14:00-15:00
- NOTIS application accessible during session-2
- UCC validation skipped during contingency periods

## Compliance Requirements

- Mandatory participation using all trading software
- Re-login into live environment post-session
- Only valid UCC/PAN entries accepted
- Contact uci@nse.co.in for UCC queries

## Summary

BSE has scheduled a mock trading session for the Equity Derivatives segment on Saturday, September 06, 2025. The session will run from 11:15 AM to 03:30 PM with log-in starting at 10:15 AM. This mock trading coincides with the release of BOLT Pro TWS version 12.03.

## Key Points

- Mock trading session scheduled for Saturday, September 06, 2025
- Available for Equity Derivatives segment only
- New BOLT Pro TWS version 12.03 will be released
- Members can choose between mock trading or UAT environment for regulatory compliance
- Feedback required by 4:30 PM on the same day
- Third party vendors and in-house system developers can test via IML/ETI APIs

## Regulatory Changes

No regulatory changes. This is a routine mock trading session in compliance with SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234 dated November 24, 2020 regarding testing requirements.

## Compliance Requirements

- Trading members must participate in either mock trading session or UAT environment
- Members using third party platforms or in-house systems should test their applications
- Mandatory feedback submission by 4:30 PM on September 06, 2025
- Reference to Exchange Notice No 20201201-22 for software testing requirements

## Important Dates

- **September 06, 2025**: Mock trading session date
- **10:15 AM - 11:15 AM**: Log-in window
- **11:15 AM - 03:30 PM**: Continuous trading session
- **4:30 PM**: Feedback submission deadline

## Impact Assessment

Medium impact event focused on system testing and preparation for new software release. Members need to allocate resources for Saturday testing session and ensure their trading systems are compatible with the new BOLT Pro TWS version. No direct market impact as this is a mock session, but important for operational readiness.

## Summary

BSE announces changes to multiple indices effective August 25, 2025, due to the amalgamation of Udaipur Cement Works Limited (530131) with JK Lakshmi Cement Ltd. (500380). Udaipur Cement Works will be dropped from various indices, while Axiscades Technologies Ltd. (532395) will be added to several indices as a replacement.

## Key Points

- Udaipur Cement Works Limited (530131) amalgamating with JK Lakshmi Cement Ltd. (500380)
- Changes effective Monday, August 25, 2025
- Multiple BSE indices affected including SmallCap, AllCap, BSE 1000, and Next 500
- Axiscades Technologies Ltd. (532395) added to four indices as replacement
- Changes included in tonight's end-of-day corporate action file (*.SDE)

## Regulatory Changes

Index composition changes due to corporate amalgamation under scheme of arrangement. No new regulatory requirements introduced.

## Compliance Requirements

- Market participants should note index composition changes for portfolio rebalancing
- Index fund managers need to adjust holdings accordingly
- Corporate action file (*.SDE) contains detailed implementation instructions

## Important Dates

- **August 25, 2025**: Effective date for all index changes
- **August 20, 2025**: Notice date and inclusion in corporate action file

## Impact Assessment

Medium impact on affected indices and index-based products. Routine corporate action with limited market disruption expected. Index funds tracking affected indices will need to rebalance portfolios. Udaipur Cement Works shareholders will receive JK Lakshmi Cement shares per amalgamation terms.
